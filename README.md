
# About the project

    It is a Web Application where in we take quiz of a topic at different levels,
    ranking and solutions with explanation is also provided. 


## Build with 
     • Operating system : windows 10
     • Technology : PHP
     • Web Technology : HTML,CSS
     • Database : MySQL
